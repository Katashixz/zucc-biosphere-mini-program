# 城院生态圈
## 简介
- 《城院生态圈》是一款基于微信小程序+SpringBoot的校内动植物百科以及论坛小程序，面向浙大城市学院范围内的师生，目前正在开发中。在这里你可以查看本校内所有动植物的个体信息，包括学名、外号、习性、生物学分类、养护指南等，也可以在里面和其他动植物爱好者交流，甚至可以云领养流浪动物，在虚拟世界中完成和流浪动物的交互。
- 本仓库属于前端模块。
